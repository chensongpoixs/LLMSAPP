



# 一、大模型

- LLM是人工智能领域的一项重大突破
- 它是一种经过海量文本数据训练的深度学习模型
- 它能够理解、生成、总结和翻译人类的语言
- 目前所有的LLM都是基于<font color='red'>Transformer</font>实现的


## 1、LLM具有多模态的能力

1. 生成文本
2. 生成图片
3. 生成视频Sora
4. 生成音频
5. 文字与音频的互转（TTS）



##2、 现在与未来

1. Scaling Law为AI带来了革命性影响
2. Scaling Law：数据量，计算量、模型越大效果越好
3. 通过RLHF让模型与人类对齐，仿佛让AI具有了人的智慧
4. 未来多模态、Agent是发展的重点方向



##  大模型的训练


大模型的训练整体上分为三阶段

1、预训练
2、SFT(监督微调)
3、RLHF(基于人类反馈的强化学习)
