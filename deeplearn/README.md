# 人工智能、机器学习与深度学习

![人工智能、机器学习、AIGC、深度学习和大模型关系图 ](/img/ai_ml_aigc_dl_llm_relation.svg)

- 人工智能： 让机器可以像人一样完成任务的智能系统
- 机器学习： 让机器从数据中学习、实现人工智能
- 深度学习： 通过多层神经网络从数据中学习的人工智能


## 一、函数是如何被找到的

1. 通过大量的数据送给深度神经网络
2. 找到每个神经元的权重（复杂的过程）
3. <font color='red'>神经网络</font> + 各神经元的<font color='red'>权重就是函数 </font>
4. 各神经元的权重的集合称为<font color='red'>模型</font>


注释：

- <font color='red'>神经网络 ：</font> 由多层神经元构成， 包括输入、输出和隐藏层
- <font color='red'>模型： </font>模型是各个神经元的<font color='red'>权重集合</font>


### 1、神经元

- 它是构成神经网络的基本单元
- 它的作用是接收、处理和传递信息


### 2、数据的种类

- <font color='red'>结构化数据： </font> 房子大小， 房间数量（数据库中的数据）
- <font color='red'>非结构化数据：</font> 图像， 音频、视频、文字
- <font color='red'>计算机</font>善于处理结构化数据， 人类善于处理非结构化数据
- 深度学习<font color='red'>是从非结构化数据中学习，</font>找到解决问题的函数

### 3、深度学习的学习策略或者训练的方法

1. 监督学习
2. 无监督学习
3. 半监督学习
4. 强化学习


#### ① 什么是监督学习？

- 每个训练样本都有标签（答案）
- 训练时，将预测值与标签进行比较
- 工具代价函数的结果调整参数权重
- 目标是用训练模型预测未见过数据的标签
- 主要应用有： <font color='red'>图像识别</font> （分类），<font color='red'>股票价格预测</font>(回归)


#### ② 无监督学习

![无监督学习](/deeplearn/unsupervised_flowchart.svg)



![无监督学习的编码和解码后数据对比的模型](/deeplearn/reconstruction_flowchart.svg)

- 无监督学习的目标是发现数据中的结构、模式和分布
- 主要用于图像去噪，社交网络分析（聚类）等场景

####  ③半监督学习

- 使用少量的标记数据和大量未标记数据进行训练
- 目标是利用未标注数据提高训练效率
- 如通过标注数据训练模型，给为标注数据打标签
- 主要应用包括文本分类， 图像识别等

####  ④ 强化学习

- 通过交互学习最佳行为或策略，如给某种奖励
- 大语言模型中的点赞就属于强化学习
- 通过与人的交互， 让大语言模型的输出更符合人的行为

#### 拟合与代价函数

- 拟合： 模拟，合成为一条线
- 损失函数： L(i)(w, b) = 1/2(y^(i) - y^(i))^2
- 代价函数：J(w, b) = 1/m(w, b)
- 结果越小， 拟合效果越好 

一条直线上的误差




![拟合与代价函数、过拟合、欠拟合](/deeplearn/over_underfit.svg)


#### 过拟合

- 过拟合： 在训练数据集上表现很好、测试集表现很差





原因1： 网络过于复杂

原因2： 数据量不够

#### 欠拟合

欠拟合： 在训练数据集上表现就很差


说明训练出模型不是最佳的模型

原因：

1. 网络复杂性太低
2. 训练数据不够
